Use the start/stop button to run the simulation. Use reset to start over. With the option AVM, the simulation follows the logics of the adaptive voter model with 100 nodes and 8 initial opinions. With the option Exploit the simulation closely follows the logics of copan:Exploit with effort levels 0.5 (sustainable )and 1.5 (non-sustainable). Use the slider to adjust the parameters. They can only be changed when the simulation is stopped.
